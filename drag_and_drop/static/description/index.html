Drag and Drop

This application allows you to drag and drop files into the attachment, message or internal log section. You can also upload screenshots.

Works with the Community and Enterprise edition but not with the OCA Version.

Features

  • Upload multi attachments to model-instance
  • Upload multi attachments to messenger
  • Upload screenshots to attachments, new message and log note with CTRL+V
